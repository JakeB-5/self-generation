sdd prompt 명령어를 실행합니다.

```bash
sdd prompt $ARGUMENTS
```

## 사용 가능한 프롬프트

- `change`: 변경 제안 작성
- `apply`: 변경 적용
- `archive`: 변경 아카이브
- `validate`: 스펙 검증

## 옵션

- `--list`: 사용 가능한 프롬프트 목록

프롬프트 내용을 출력하고 사용 방법을 안내해주세요.
